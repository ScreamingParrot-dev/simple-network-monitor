# simple-network-monitor
A simple GO network node monitoring agent.

The utility simulates the operation of an OAM system. It polls URL addresses specified in a text file using various protocols (ICMP/Ping, TCP, HTTP) and collects availability metrics.

Простой агент мониторинга сетевых узлов на GO. 

Утилита имитирует работу системы управления (OAM). Она опрашивает заданные в текстовом файле URL адреса по разным протоколам (ICMP/Ping, TCP, HTTP) и собирает метрики доступности.
